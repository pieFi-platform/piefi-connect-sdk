## 🔗 Installation
This is a server side npm package can can be installed: 
```
npm install --save @piefi/piefi-connect-sdk
```

## ⚡️Quick Start
```
import PieFiConnect from "@piefi/piefi-connect-sdk";
const connect = new PieFiConnect('API_KEY', 'COMPANY_ID');
```
